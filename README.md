# VERO-Digital-Solutions (Python Task)

Clone the repository:

```
git clone git@github.com:Farvakhursheed/VERO-Digital-Solutions.git
```

Install requirements:

```
pip3 install -r requirements.txt
```

To run server code:

```
cd app
python3 app.py
```

To run client code:

```
cd client
python3 client_script.py
```
